# Project-Specific Conventions

This directory contains project-specific conventions, vision files, and templates discovered or configured by Shipmate.

## Vision Files

- `mission.md` - Product mission and target users
- `architecture.md` - System architecture and technology stack
- `domain.md` - Domain model and business rules

## Convention Files

- `pull-requests.md` - PR conventions (title format, description template, review requirements)
  - Generated by `/shipmate-learn` command
  - Used by `/shipmate-ship` to create consistent PRs

## How to Update

Run `/shipmate-learn` to re-analyze and update project conventions.

Manual edits to these files will be preserved during re-analysis (you'll be prompted to confirm changes).
